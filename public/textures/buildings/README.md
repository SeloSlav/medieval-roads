# Building texture sources

The texture maps in this directory are 1K assets downloaded from
[Poly Haven](https://polyhaven.com/) and used under
[CC0 1.0](https://polyhaven.com/license). They may be used, modified, and
redistributed without attribution. Source links are retained here for
provenance and future updates.

Only diffuse, OpenGL normal, and roughness maps are shipped. Displacement and
ambient-occlusion maps are intentionally omitted to keep the building material
set lightweight and compatible with low-poly geometry.

| Local prefix | Poly Haven asset | Source |
| --- | --- | --- |
| `plaster` | Plastered Stone Wall | https://polyhaven.com/a/plastered_stone_wall |
| `masonry` | Medieval Wall 02 | https://polyhaven.com/a/medieval_wall_02 |
| `clay_tiles` | Clay Roof Tiles | https://polyhaven.com/a/clay_roof_tiles |
| `wood_planks` | Wood Planks | https://polyhaven.com/a/wood_planks |
